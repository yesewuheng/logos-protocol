# MMU (最小记忆单元) 规范 v1.2

MMU 是 Logos Protocol 的遗传载体。只记录失败，不记录成功。

## 格式规范

每个 Failure Snapshot 必须包含：
- **Cycle**：编号
- **场景**：发生环境
- **问题**：具体失败描述
- **代价**：实际损失（金钱/时间/信任）
- **Constraint**：强制影响下一代行为的约束
